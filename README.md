# macgyver-frontend
Frontend app for https://github.com/navikt/macgyver that fixes stuff like macgyver for team sykmelding

## Getting Started

### install dependencies
```bash
yarn
```

### build
```bash
yarn build
```

### run development server

```bash
yarn start
```
